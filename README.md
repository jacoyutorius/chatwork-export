## Chatwork Export


## Install

```
git clone git@bitbucket.org:FOURIER_INC/chatwork-export.git
cd chatwork-export
bundle install
```

## Usage

```
bundle exec ruby export.rb #{your chatwork email} #{your chatwork password}
```
